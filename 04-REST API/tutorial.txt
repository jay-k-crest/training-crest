        -Support for Filtering (e.g., /courses?courseType="engineering")

STEP 19 : Identify the need for pagination and add if needed
GET /courses
Request :
   page:
    -support for pagination (e.g., /courses?page=2)
   size : 
    -support for pagination (e.g., /courses?size=10)
    both :
    -support for pagination (e.g., /courses?page=2&size=10)
